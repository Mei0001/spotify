Spotify Weather MCP

「今日の気分、音楽にして」- それだけでOK。

Spotify認証

Client IDは32文字の英数字です

Client IDは Spotify Developer Dashboard で取得できます

セットアップ手順

  1. Spotify Developer Dashboard でアプリを作成し Client ID を取得
  2. アプリのRedirect URIに を追加
  3. リポジトリをクローン: git clone https://github.com/Mei0001/spotify.git
  4. npm install
  5. npm run auth を実行（ローカルの受信サーバーが起動＆このページが開きます）
  6. 上のフォームにClient IDを入力して認証
  7. npm run dev でMCPサーバーを起動
  8. VoiceOSで接続！
